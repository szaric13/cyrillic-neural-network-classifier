# Cyrillic Neural Network Classifier

This project implements a fully connected neural network from scratch using Python and NumPy to classify the first 10 Cyrillic letters (А–И).

The model is built without any machine learning frameworks such as TensorFlow or PyTorch, focusing on a deep understanding of core neural network concepts.

## Key Features

• Neural network implemented from scratch (NumPy only)
• Forward and backward propagation
• Cross-entropy loss function
• Gradient-based optimization
• Image preprocessing (28×28 grayscale → 784 vector)
• Multi-class classification (10 Cyrillic letters)

## Dataset

Custom dataset of handwritten Cyrillic letters organized into 10 classes:
А, Б, В, Г, Д, Ђ, Е, Ж, З, И

## Tech Stack

Python, NumPy
